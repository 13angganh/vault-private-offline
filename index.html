Setup Vault Baru
Ikuti 3 langkah berikut
1
Buat Master Password
Password ini digunakan untuk mengenkripsi semua datamu. Jangan sampai lupa — tidak ada server yang bisa memulihkannya.
2
Recovery Phrase — Catat & Simpan!
5 kata acak ini adalah satu-satunya cara reset master password jika lupa. Simpan di tempat berbeda (kertas, notes HP, dsb). Tidak bisa dilihat lagi setelah setup.
3
Konfirmasi
Pastikan semua data di atas sudah benar sebelum membuat vault.
